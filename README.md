# 智能待办网站 - Vue前端

一个基于Vue 3开发的智能待办事项管理网站，支持自然语言输入、日期智能识别、任务统计等功能。

## 功能特性

### 🎯 核心功能
- **自然语言任务添加**：支持"25号上午十点 测试1"等自然语言输入
- **智能日期识别**：自动识别日期和时间，支持相对日期、星期、具体日期等格式
- **日期验证提醒**：当设置日期在当前月份已过时，提示用户是否顺延到下个月
- **任务管理**：添加、编辑、删除、完成任务
- **分类查看**：所有任务、已完成任务、已逾期任务

### 📊 统计功能
- **总体统计**：总任务数、已完成、待完成、已逾期
- **完成率图表**：环形进度图展示完成率
- **本周统计**：按天展示本周任务完成情况
- **个人界面集成**：通过个人界面的完成率卡片跳转到详细统计

### 🎨 界面设计
- **响应式设计**：完美适配手机和电脑端
- **底部导航**：任务列表、日历、个人三个主要功能入口
- **现代化UI**：简洁美观的用户界面

## 技术栈

- **前端框架**：Vue 3 + Composition API
- **路由管理**：Vue Router 4
- **构建工具**：Vite
- **日期处理**：Day.js
- **样式**：原生CSS + 响应式设计

## 安装和运行

### 环境要求
- Node.js 14+
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 开发模式运行
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 使用说明

### 1. 添加任务
- 在任务列表页面的输入框中输入自然语言，如：
  - "明天上午九点开会"
  - "25号下午三点测试项目"
  - "下周一完成报告"

### 2. 日期验证
- 如果设置的日期在当前月份已过（如今天29号，设置25号），系统会提示是否顺延到下个月

### 3. 查看统计
- 进入个人界面，点击完成率卡片查看详细统计信息
- 统计页面包含总体统计、完成率图表和本周统计

### 4. 移动端访问
- 项目支持移动端访问，可通过网络地址在手机上查看

## 特色功能

### 自然语言日期解析
支持多种日期格式：
- 相对日期：今天、明天、后天、昨天等
- 星期：周一、周二、周三等
- 具体日期：25号、10-25、2024-10-25等
- 时间：上午十点、10:30、下午三点等

### 智能日期处理
- 默认使用当前月份
- 日期已过时自动顺延到下个月
- 无明确日期时默认设为当天

## 开发说明

### 日期解析器
项目包含强大的自然语言日期解析器，位于 `src/utils/dateParser.js`，支持复杂的日期时间识别。

### 响应式设计
使用CSS媒体查询和弹性布局，确保在各种设备上都有良好的用户体验。
